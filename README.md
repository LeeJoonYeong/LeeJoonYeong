  저는 웹퍼블리셔와 프론트엔드를 좋아하고 일을 하고있는 개발자입니다. (그냥 컴퓨터를 좋아합니다 :) )<br>

  그렇게 처음 만난 언어는 C++입니다. 저를 컴퓨터 세계에 입문하게 해준 고마운 녀석입니다. ^^<br>
